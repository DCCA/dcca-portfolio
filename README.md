# Daniel Andrade portfolio

Personal portfolio for Daniel Andrade, focused on AI-native product systems, signal curation, eval gates, and human review loops.

## Stack

- React
- TypeScript
- Vite
- Bun
- Native CSS

## Run locally

```bash
bun install
bun run dev
```

## Verify

```bash
bun run build
```

## Notes

This repo is the source for a static personal portfolio. Generated production files live in `dist/` and are intentionally not committed.
